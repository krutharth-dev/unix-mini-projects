# Unix Mini Projects

A collection of Unix shell scripting mini projects created for learning and demonstrating practical Unix/Linux automation tasks.

## Projects

1. Process Monitoring and Alert System
2. Interactive Menu-Driven System Script
3. Simple Backup/Restore Manager
4. Disk Usage Email Alert
5. ASCII Art File Converter

## Current Project Included

### 01 - Process Monitoring and Alert System

A shell script that checks whether selected processes are running, logs their status, and optionally alerts or restarts stopped services.
